Orchestrator '85 is a Windows desktop utility for managing and launching multiple applications from a single control panel.

You define a list of apps — each with an exact window position, size, monitor, startup delay, and launch arguments. Hit "run" and Orchestrator places every app exactly where you want it on your screen(s), automatically. Hit "stop" to close them all.

Key features:

App list — add executables, batch files, or shortcuts via drag-and-drop or file browser

Window positioning — set exact X/Y/W/H per app, per monitor; visual preview included

Profiles — switch between named sets of apps (e.g. "work", "gaming", "streaming")

Scheduler — auto-launch apps on a schedule or at startup

Global hotkeys — bind keyboard shortcuts to run/stop individual apps or all at once

System monitor — real-time CPU/GPU temperature, load, RAM and VRAM usage

Watchdog — automatically restart a crashed application

Localhost launcher — open up to 10 local dev server ports in the browser with one click

Tray icon — runs quietly in the background, accessible from the system tray

Portable — single .exe, no installer required; stores all data next to itself

Built with a retro Windows '85 aesthetic. Designed for power users, developers, and streamers who want full control over their desktop layout.
